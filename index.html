Projects

django-responsive-tables2

This Django package is intended to add functionality to the already outstanding django_tables2 library. Using this package, you'll be able to easily implement responsive full-text search capabilities, pagination and more. Although a default bootstrap template is included, this library allows for customization, and a template can be used to theme the table however required.

edge-runlist-builder

This tool automates building vehicle runlists for the Edge Pipeline auction sales platform. This tool uses Selenium to pull presale auction data to match with owned units on the point of sales for easy calculation of breakeven costs, reserve/line prices, source of purchasing and more. This tool copies the table it generates into the clipboard, so it can be easily entered in Microsoft Teams or an Excel document.

sonos-controller

This app serves as a controller for your Sonos multiroom audio system using Python. Using sonos-controller you'll be able to control the volume of your Sonos zones using the native media keys on your keyboard. Hotkeys can be changed, and players can be added or removed by modifying the XML configuration file.

kijiji-scraper

This scraper is a web crawler which provides a way to receive new listing notifications sooner than the ones from Kijiji which are only sent daily (once every 24-hours). This scraper also increases the functionality with bump and price change notifications. Notifications are sent using Pushover, and include a title, short description, image, price, phone number (if applicable) and a link to the ad.
